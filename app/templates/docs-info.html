This system contains important family documents and history. The documentation explains how everything works so the system can continue to be maintained.

Key points:

  • All original documents are stored in Cloudflare R2 (see .env file for credentials)
  • SQLite database is in the project directory (life_admin.db)
  • Backups are in the backups/ directory
  • The system is designed to last 10-20+ years
  • Everything is documented in the docs/ folder

Read docs/00-vision-and-principles.md first to understand the philosophy, then docs/99-build-log.md for technical details.
